> GDD v5.0 기준 | (주)문래컴퍼니 | 2026년 4월

## 게임 핵심 컨셉

**방치형 펫 육성 RPG + 던전 공략 + 만보기 연동 위치기반 수집형 게임**

- 알에서 부화한 100종 이상의 달리몬을 키우며 던전을 정복하는 방치형 RPG
- 걸음수(만보기)·GPS 구역이 핵심 자원 — 일상과 게임이 자연스럽게 연결됨
- **핵심 장면**: 할아버지-손자 레벨 격차 배틀에서 약자 핸디캡 보정으로 함께 즐기기
- 모든 연령 타깃 — "설명 없이 3분 안에 시작, 글 몰라도 아이콘만으로 플레이"

### 투트랙 구조

| 트랙 | 이름 | 역할 |
|------|------|------|
| 트랙 1 | 주인공 (캐릭터) | 일상 속 걷기 친구·정서적 애착 대상 — 만보 EXP 수집, 달리몬 레벨 상한 결정 |
| 트랙 2 | 달리몬 | 전투 펫 — 던전/배틀/5v5 PvP, 레벨 상한 = 주인공 레벨 |

두 트랙은 연결: 걸음 → 던전 입장권 → 달리몬 레벨업 / PvP 승리 → 주인공 코스튬 드롭

## 달리몬 시스템

### 7가지 타입

| 타입 | 아이콘 | GPS 보너스 구역 | 배틀 특성 | 스탯 강점 | 약점 타입 |
|------|--------|-----------------|-----------|-----------|-----------|
| 대지 | 🌍 | 산·자연·숲길·조용한 주거지역 | 반격 데미지 | DEF +40%, HP 회복↑ | 식물 |
| 불 | 🔥 | 도심·상업지구·학교 근처 | 선공 확률 높음 | ATK +30% | 물 |
| 물 | 💧 | 강변·해변·수변 공원 | HP 회복·지속 데미지 | HP +30%, 회복력↑ | 대지 |
| 바람 | 💨 | 공원·광장·하천변·운동장 | 회피·카운터 강함 | SPD +25%, 회피↑ | 불 |
| 식물 | 🌿 | 숲·정원·식물원 | 지속 성장·독 효과 | ATK·DEF 균형 | 바람 |
| 어둠 | 🌑 | 야간 전용 (20:00~06:00)·지하도·골목 | 상태이상 유발 | 크리티컬 +20% | 빛 |
| 빛 | ✨ | 광장·명소·랜드마크 | 아군 버프·회복 | 전체 스탯 +10% 팀 버프 | 어둠 |

GPS 구역 내에서 해당 타입 전 스탯 +20% 보너스.  
어둠 타입: 야간에 EXP +30%, 낮에는 GPS 보너스 미적용.

### 타입 상성

```
물 → 불 → 식물 → 대지 → 바람 → 물  (순환)
빛 ↔ 어둠
강점 타입 상대 시 데미지 ×1.5 / 약점 타입 상대 시 ×0.7 / 중립 ×1.0
```

### 등급 및 희귀도

| 등급 | 종류 수 | 배경색 | 획득 방법 |
|------|---------|--------|-----------|
| 일반 (Common) | 60종 | 회색 | 일반 알 (200보) |
| 희귀 (Rare) | 30종 | 파란색 | 희귀 알 (1,000보 or 던전 Lv.10+) |
| 전설 (Legendary) | 10종 | 금색 | 전설 알 (5,000보 or 이벤트) |

### 알 시스템

| 알 등급 | 부화 비용 | 획득 방법 | 결과 |
|---------|-----------|-----------|------|
| 일반 알 (흰색) | 200보 | 기본 지급·던전 드롭 | 일반 등급 (전 타입) |
| 희귀 알 (금색) | 1,000보 or 던전 Lv.10+ 드롭 | Lv.10 던전·주간 퀘스트 | 희귀 등급 (전 타입) |
| 전설 알 (은하색) | 5,000보 or 특별 이벤트 | 시즌 이벤트·프리미엄 뽑기 | 전설 등급 (전 타입) |
| 타입 전용 알 (7종) | 던전 클리어 보상 or 타입 구역 탐험 | 해당 타입 GPS 구역·전용 던전 | 해당 타입 달리몬만 부화 |

### 배틀 스탯 공식

```
ATK = 기본ATK + 레벨×2 + 누적걸음수×0.001
DEF = 기본DEF + 레벨×1.5 + 던전클리어횟수×5
HP  = 200 + 레벨×10 + 만보달성횟수×5
SPD = 기본SPD + 레벨×1 + 연속걷기일수×2
크리티컬 = 레벨×0.2% (최대 20%, 어둠 타입 +10% 추가)
GPS 보너스 = 자기 타입 구역에서 전 스탯 +20%
```

### 진화 테이블

| 레벨 | 단계명 | 배틀 능력 해금 | 비고 |
|------|--------|----------------|------|
| Lv.0 | 알 (Egg) | 없음 | 부화: 200~5,000보 |
| Lv.1~9 | 기초형 | 없음 | 방치 EXP 획득 시작 |
| Lv.10~24 | 1진화 | 기본 공격+가드 | 던전 Lv.10 입장 가능 |
| Lv.25~49 | 2진화 | 스킬 2개+카운터 | 던전 Lv.25 입장 가능 |
| Lv.50~79 | 3진화 | 스킬 3개+필살기 | 던전 Lv.50 입장 가능 |
| Lv.80~99 | 최종형 | 스킬 4개+팀 버프 | 던전 최종 보스 입장 |
| 프레스티지 | 프레스티지형 | 전 스킬+보너스 | 레어 코스튬·신규 알 해금 |

### 달리몬 관리

- **파티**: 던전용 최대 3마리. 같은 타입 3마리 +15%, 상성 좋은 2종 이상 데미지 +10%, 빛+어둠 조합 크리티컬 +15%
- **합성**: 같은 종 3마리 합성 → 희귀도 1단계 상승
- **해방**: 불필요한 달리몬 해방 → EXP 보상
- **슬롯**: 기본 5슬롯 → 던전 클리어·퀘스트로 최대 30슬롯 확장

## 던전 시스템

### 던전 유형

| 유형 | 특징 | 입장 방법 | 보상 |
|------|------|-----------|------|
| 일반 던전 | Lv.1~∞ 순차 클리어 PvE | 홈 아이콘 → 레벨 선택 → 자동 배틀 | EXP·알·아이템 |
| 지도 던전 (GPS) | 실제 지도 기반, 야생 달리몬 조우 & 근처 유저 PvP | 홈 아이콘 → 지도 던전 → 타겟 선택 | 지역 특화 알·야생 EXP |
| 타입 던전 | 특정 타입 달리몬만 입장, 주 1회 갱신 | 타입 조건 충족 시 해금 | 타입 전용 알·코스튬 |
| 보스 던전 | 특정 Lv. 도달 시 해금, 주 1회 | 레벨 조건 충족 시 해금 | 전설 아이템 확정 |
| 야간 던전 | 20:00~06:00만 개방, 어둠 타입 전용 | 야간 시간대 | 어둠 타입 알·야간 코스튬 |
| 이벤트 던전 | 시즌 한정, 특별 달리몬 등장 | 시즌 이벤트 기간 | 시즌 전용 아이템 |

- 배틀 형식: 3~5연전 자동 배틀 (결과만 확인)
- 입장 비용: 던전 입장권 (만보기·퀘스트로 획득) or 광고 시청
- 보스 스테이지: +300% EXP 보너스

### NFC 용병 시스템

오프라인에서 스마트폰을 맞대어(NFC 태그) 친구·보호자의 달리몬을 임시 대여.  
1회 태그 시 1시간 유지. 고난이도 일반 던전·보스 던전에 용병으로 편성 가능.  
초보 유저 허들 완화 + 오프라인 바이럴·가족/친구 소통 강화 목적.

## 소셜 & PvP 시스템

### 지도 던전 친구 레이더

GPS 기반 지도 던전 화면에 실시간으로 주변 친구 달리몬 표시.  
50m 이내 접근 시 근접 알림 발생. 친구 아이콘 탭 → EXP 소량 주고받기 가능.

### 5v5 PvP 배틀

| 구분 | 내용 |
|------|------|
| 진입 | 지도 던전에서 근처 유저 탭 후 '배틀 신청' or 오프라인 NFC 접촉 |
| 덱 구성 | 최대 5마리 총력전 |
| 배틀 룰 | 자동 전투, 타입 상성 시너지 + 5마리 팀 버프 극대화 |
| 보상 | PvP 랭킹 포인트 + 승리 토큰 (한정 상점 교환용) |

## 핵심 설계 원칙

### 5원칙

| 원칙 | 내용 |
|------|------|
| 3탭 원칙 | 모든 핵심 기능은 홈에서 3번 탭 이내 도달 |
| 텍스트 최소화 | 글 대신 아이콘+숫자. 설명 없어도 직관적으로 |
| 자동 기본 | 방치형 EXP·배틀 모두 자동 진행. 조작은 선택사항 |
| 즉각 반응 | 걸음마다 숫자 증가, 레벨업 시 진동+소리 |
| 실패 없음 | 던전 실패해도 EXP 획득, 배틀 져도 성장 |

### 핵심 불변 규칙

- **만보 = 핵심 자원** — 걸음수가 EXP·입장권·알 부화의 공통 원천
- **펫 레벨 상한 = 주인공 레벨** — 주인공(캐릭터)을 키워야 달리몬도 강해짐
- **약자 핸디캡 보정** — 레벨 격차 클수록 약자에게 보정. "할아버지-손자 편먹기" 보장
- **타입 상성**: 물→불→식물→대지→바람→물 / 빛↔어둠 (강점 ×1.5, 약점 ×0.7)

## 배틀 핵심 로직

```dart
// 핸디캡 보정 (레벨 차이 기반)
// diff ≤ 5  → 보정 없음
// diff ≤ 15 → 약자 +15%
// diff ≤ 30 → 약자 +25%
// diff > 30 → 약자 +35%

// 감투 발동: 약자 HP 30% 이하 + 60% 확률 → 다음 턴 ATK +30%

// 배틀 종료 조건: 어느 쪽 HP ≤ 0 OR 12턴 초과 → HP 높은 쪽 승리
// 패배해도 EXP 지급 (실패 없음 원칙)
```
